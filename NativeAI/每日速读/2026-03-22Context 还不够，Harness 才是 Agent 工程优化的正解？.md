2. Harness engineering 已经在解决结构性问题了，而非仅仅是上下文的问题。
3. 人类可以控制的部分应该逐渐变少，AI的潜力已经大于人类的能力了，人类应该作为 Reward Model 存在，去训练 Agent 和 AI，或者是说大语言模型更贴合人类的偏好。
