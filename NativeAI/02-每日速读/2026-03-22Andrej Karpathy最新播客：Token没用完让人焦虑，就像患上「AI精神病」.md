**一句话概括**：Agent是一种新的操作系统
**我的判断**：
1. Agent会改变原有的操作系统，一切API化就可以，但是安全问题、责任归属问题如何解决呢？
2. 人类的工作从细粒度变为粗粒度，现在利用AI要有全局视角和下棋思维，如何编排好流程和系统的价值由于你自己能做好单点任务；所以人的能力到底体现在哪来呢？
3. 人类对情感的需求是不可替代的；比如codex在笔者眼里更好，是因为他有人格和性格；虽然是设定的；
4. 【产品视角】Tokens可视化和合理利用与转移是不是一个方向呢？

# 高亮语句
1. 新范式：以 Agent 为核心的软件生产方式重构。
2. 设备只需开放 API，Agent 会成为新的「操作系统」
3. 一个研究机构，本质上就是一组 markdown 文件 —— 角色、流程、协作方式，全都是「代码」；
4. 而是开始思考，如何同时运行多个 Agent，如何去调度它们、组织它们，让它们真正协同工作
5. 而是你还没找到一种方法，把现有的一切串联起来。
6. 你开始用更大粒度的宏操作在行动。
7. 没把 token 的吞吐量用到最大，有些焦虑
8. 反而像 Claude 的人格就做得挺不错，更像一个队友，会和你一起兴奋、一起投入。
